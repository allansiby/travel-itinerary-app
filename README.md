# Travel Itinerary Application

This is a frontend-only Travel Itinerary Application built using Angular 19.

## Features

- Destination-based travel package listing
- Add packages to cart using localStorage
- Cart summary with total cost
- Modern Angular 19 control flow syntax

## Technology Stack

- Angular 19
- TypeScript
- HTML & CSS

## How to Run the Project

1. Install dependencies  
   npm install

2. Run the application  
   ng serve

3. Open in browser  
   http://localhost:4200
